# EShop

<p>数据库：MySQL</p>
<p>需要通过 python3 安装工具进行安装依赖，</p>
`create database hero;`
```
pip3 install -r requirements.txt
```
<p>`requirements.txt`中包含</p>
<p>Django==2.2.2</p>
<p>Pillow</p>
<p>python-alipay-sdk</p>
<p>django-mysql</p>
<p>PyMySQL</p>
<p>django-restframework</p>
<p>pytz</p>
<p>启动时先给run.sh 赋予权限</p>
```
chmod a+x run.sh
```
<p>然后之间运行 run.sh </p>
```
./run.sh
```
<p>服务器运行日志打印在 `nohup.out` 中</p>
